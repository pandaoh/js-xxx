# Function: data2Arr()

> **data2Arr**(`sourceData`, `key`): `any`[]

数组对象转数组值

## Parameters

• **sourceData**: `object`[]

源数据

• **key**: `string`

取值的 key

## Returns

`any`[]

## Example

```ts
data2Arr([{a: 1, b: 2}, {a: 3}, {b: 1}], 'a'); /// [1, 3]
```
