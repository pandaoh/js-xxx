# Function: addSpace()

> **addSpace**(`str`): `string`

强制给字符串添加空格间隔

## Parameters

• **str**: `string` \| `number`

字符串

## Returns

`string`

## Example

```ts
addSpace('test'); /// 't e s t'
addSpace(null); /// ''
addSpace('123 45'); /// '1 2 3 4 5'
```
