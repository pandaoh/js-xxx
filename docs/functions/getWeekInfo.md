# Function: getWeekInfo()

> **getWeekInfo**(`n`): `object` \| `object`[]

## Parameters

• **n**: `string` \| `number`

值

## Returns

`object` \| `object`[]

## Example

```ts
getWeekInfo(); /// WEEKS_INFO
getWeekInfo('0'); /// WEEKS_INFO

getWeekInfo('unknown'); /// WEEKS_INFO
getWeekInfo(1); /// 星期一相关信息
```
