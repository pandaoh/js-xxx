# Function: getStyleByName()

> **getStyleByName**(`element`, `name`): `any`

获取任一元素的 style 任意属性

## Parameters

• **element**: `any`

指定元素

• **name**: `any`

属性名称

## Returns

`any`

## Example

```ts
getStyle(document.getElementById('test'), 'width'); /// #test 的 width 属性
```
