# Function: getAge()

> **getAge**(`birthday`, `targetDate`?): `number`

获取年龄

## Parameters

• **birthday**: `any`

生日

• **targetDate?**: `any`

目标日期

## Returns

`number`

## Example

```ts
getAge('1998-9-28'); /// 24
getAge('1998-6-8', '2023-7'); /// 25
getAge('2023'); /// -1
```
