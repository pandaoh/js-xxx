# Function: getLastVar()

> **getLastVar**(`data`): `any`

获取对象/数组最后一个值(对象为 Object.keys 的顺序)

## Parameters

• **data**: `any`

源数据

## Returns

`any`

## Example

```ts
getLastVar([1, 2]); /// 2
getLastVar({a: 2, b: 1}); /// b
```
