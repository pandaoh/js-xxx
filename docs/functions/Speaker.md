# Function: Speaker()

> **Speaker**(`text`, `lang`, `volume`, `pitch`, `rate`): `object`

TTS 语音，可以在现代浏览器直接运行。

## Parameters

• **text**: `string`

内容

• **lang**: `string` = `'zh-CN'`

语言

• **volume**: `number` = `1`

音量 [0, 1]

• **pitch**: `number` = `1`

音高 [0, 2]

• **rate**: `number` = `1`

速度 [0.1, 10]
