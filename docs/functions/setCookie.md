# Function: setCookie()

> **setCookie**(`key`, `value`, `expires`): `void`

Set Cookie

## Parameters

• **key**: `string`

Cookie key

• **value**: `any`

Cookie value

• **expires**: `number`

Cookie expires

## Returns

`void`

## Example

```ts
setCookie('name', 'value', 1); /// 设置 name 对应的 Cookie 值
```
