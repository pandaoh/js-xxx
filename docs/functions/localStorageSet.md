# Function: localStorageSet()

> **localStorageSet**(`key`, `value`): `boolean`

设置 localStorage

## Parameters

• **key**: `string`

存储 key

• **value**: `any`

存储 value，value === null|undefined 清除数据。

## Returns

`boolean`

## Example

```ts
localStorageSet("key", "value"); /// 存储时不需要处理数据，value === null|undefined 清除数据。(boolean)
```
