# Function: findParents()

> **findParents**(`element`, `n`): `any`

找元素的第 n 级父元素

## Parameters

• **element**: `any`

指定元素

• **n**: `number`

第几个

## Returns

`any`

## Example

```ts
findParents(document.getElementById('test'), 3); /// #test 的第三个父元素
```
