# Function: getTimezone()

> **getTimezone**(): `string`

获取当前时区的标准格式表示。

## Returns

`string`

## Example

```ts
// 如果时区为 UTC+8
getTimezone(); // '+8'
// 如果时区为 UTC-5
getTimezone(); // '-5'
// 如果时区为 UTC
getTimezone(); // '+0'
```
