# Function: findChildren()

> **findChildren**(`element`): `any`[]

找元素的所有子元素，解决浏览器兼容问题。

## Parameters

• **element**: `any`

指定元素

## Returns

`any`[]

## Example

```ts
findChildren(document.getElementById('test')); /// #test 的所有子元素数组
```
