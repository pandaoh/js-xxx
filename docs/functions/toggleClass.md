# Function: toggleClass()

> **toggleClass**(`element`, `className`): `void`

给元素添加/删除类

## Parameters

• **element**: `any`

元素

• **className**: `string` \| `string`[]

类

## Returns

`void`

## Example

```ts
toggleClass(myElement, 'active'); /// 给元素添加/删除 active 类
toggleClass(myElement, ['active', 'disabled']); /// 给元素添加/删除 active/disabled 类
```
