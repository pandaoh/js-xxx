# Function: markNumber()

> **markNumber**(`value`): `string`

## Parameters

• **value**: `number`

数值

## Returns

`string`

## Example

```ts
markNumber(10000123111); /// '100.00 亿'
markNumber(12345); /// '1.2 万'
markNumber(123); /// '123'
```
