# Function: insertAfter()

> **insertAfter**(`newElement`, `targetElement`): `void`

## Parameters

• **newElement**: `any`

某元素

• **targetElement**: `any`

指定元素

## Returns

`void`

## Example

```ts
insertAfter(document.getElementById('a'), document.getElementById('b'));
```
